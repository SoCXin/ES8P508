## [ES8P508简介](https://github.com/SoCXin/ES8P508/wiki)

[ES8P508](https://github.com/SoCXin/ES8P508)拥有 6 路 UART 通信接口，128K+24K 存储方案，40mA电流IO

[![sites](docs/ES8P508.png)](http://www.essemi.com/product/59.html)

#### 关键特性

* UART x 6
* 宽电压 2.2～5.5V
* 128位 AES
